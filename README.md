# neve

> NEVE vocal chain with single-knob intensity control

## Controls

- **E1** — intensity (0–100, morphs through preamp → comp → soothe → air)
- **K1** — toggle clip mode (tube / tape)
- **K2** — toggle mono sum
- **K3** — bypass chain

## Grid

None (parameters animate based on intensity).

## Requirements

- norns
- engine: Neve (custom SC engine)

## Install

```
;install https://github.com/jamminstein/neve
```
